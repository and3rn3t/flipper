# AI History Logging

This folder stores prompt and response summaries for AI-assisted work in this repo.

Goals:

- Keep a lightweight, searchable trail of AI use.
- Link prompts to code changes (commits or PRs).
- Make reviews faster and more transparent.

## How to log

1. Copy the template from template.md.
2. Save it as docs/ai-history/YYYY-MM-DD\_<short-title>.md.
3. Add links to related PRs or commits.

Keep logs concise and focused on decisions and outcomes.
